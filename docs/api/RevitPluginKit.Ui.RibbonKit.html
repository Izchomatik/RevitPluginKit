AddRibbonTab(UIControlledApplication, String, List<RibbonPanelSettings>, Boolean)

Adds a revit plug-in ribbon tab.

You can call this method multiple times to add multiple tabs.

The tab will be added at the end of the main list of Revit tabs (such as Architecture, Structure, and the like). Also note that the base "Modify" tab will naturally appear behind the custom tab created by the user.
